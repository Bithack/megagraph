# MegaGraph

A GPU accelerated tool for visualizing large data sets.

## Dependencies

Before compiling, please make sure you have the following dependencies installed:


```
libglfw3
libvips
```


### Installing dependencies on OS X

```
$ brew install glfw vips libxml2
```
